# Directory

```
| - / (root)
    | - src (source, required)
    | - dev (develop)
    | - dist (production)
    | - images (image files)
    | - fonts (font files)
    | - data (mock api)
    | - media (audio, video, ...)
    | - resources (other)
    | - ...
```

* `src`: Source directory. Source codes directory, like `html, css, less, js` 
* `dev`: Developing directory. Directory to debug in browser. 
* `dist`: Production directory. Generated after building.
